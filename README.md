# Wetube

youtube clone project

## Router Setting

### Global Router

| Router      | Description     |
| ----------- | --------------- |
| **/**       | Homepage's main |
| **/join**   | Join            |
| **/login**  | Login           |
| **/search** | Search video    |

### /users

| Router      | Description             |
| ----------- | ----------------------- |
| **/edit**   | Edit user's information |
| **/delete** | Delete user's account   |

### /videos

| Router               | Description                 |
| -------------------- | --------------------------- |
| **/watch**           | Watch video                 |
| **/edit**            | Edit video                  |
| **/delete**          | Delete video                |
| **/comments**        | Comment on a video          |
| **/comments/delete** | Delete a comment of a video |
