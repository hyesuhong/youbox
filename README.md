# Youbox

youtube clone project

## Router Setting

### Global Router

| Router      | Description     |
| ----------- | --------------- |
| **/**       | Homepage's main |
| **/join**   | Join            |
| **/login**  | Login           |
| **/search** | Search video    |

### /users

| Router      | Description         |
| ----------- | ------------------- |
| **/logout** | Log out             |
| **/:id**    | See user's profile  |
| **/edit**   | Edit My information |
| **/delete** | Delete My account   |

### /videos

| Router          | Description  |
| --------------- | ------------ |
| **/:id**        | Watch video  |
| **/:id/edit**   | Edit video   |
| **/:id/delete** | Delete video |
| **/upload**     | Upload video |
